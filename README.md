# SENTINEL Go Engine
โปรเจกต์อัปเกรดระบบรักษาความปลอดภัย (SENTINEL) จาก PHP เป็นภาษา Go

## ความสามารถของระบบ (Features)
- High-speed Security Scanner (Go Concurrency)
- AI Bridge for Web Security
- Monitor Domain Health & Email Security

## เทคโนโลยีที่ใช้
- Language: Go (Golang)
- Integration: PHP / MySQL
